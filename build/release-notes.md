# a8a 업데이트 안내

이번 버전은 DATA OUTPUT 표현식과 환경 실행 경고를 추가하고, API 실행 증적과 스크립트 콘솔 가독성을 정리한 Windows 릴리즈입니다.

- DATA 모듈 OUTPUT 문자열에서 `[[ ]]` 표현식으로 현재 INPUT 값을 참조할 수 있습니다.
- `[[amount * 10]]`처럼 표현식이 문자열 전체를 차지하면 숫자, 불리언, 객체 타입을 보존합니다.
- BASE가 아닌 환경에는 `실행시 경고표시` 옵션을 켤 수 있으며, 전체 캔버스 실행, 모듈 INPUT 미리보기 실행, API 테스트 실행 전에 현재 환경명을 확인합니다.
- 경고창에서 `경고를 표시하지 않음`을 체크하고 확인하면 해당 환경의 실행 전 경고 옵션이 해제됩니다.
- API 모듈 실행 로그는 실제 요청 URL, Header, Body, 응답 Body, cURL 명령을 노드 실행 단위로 기록합니다.
- 스크립트 콘솔과 실행 로그 콘솔은 공통 테마 색상을 사용해 밝은/어두운 화면에서 모두 읽기 쉽게 표시됩니다.
- README와 아키텍처 문서에 DATA OUTPUT 표현식, 환경 실행 경고, 실행 계약을 반영했습니다.
