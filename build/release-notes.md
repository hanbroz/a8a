# a8a 업데이트 안내

이번 버전은 **세션 상태 유지** 기능을 추가한 Windows 릴리즈입니다.

- 앱을 종료했다 다시 켜도 **마지막 실행 로그·노드 결과·End 표시값**이 그대로 복원됩니다. 값을 다시 조회하거나 API를 다시 호출할 필요가 없습니다.
- 마지막에 **열려있던 프로젝트와 화면(뷰·로그 패널)** 도 그대로 이어서 열립니다.
- 저장되는 실행 결과에는 API 응답이 포함될 수 있어, 파일을 **OS 키체인으로 암호화**하여 보관합니다.
- 종료 당시 진행 중이던 실행의 잔여 상태(멈춘 진행 표시 등)는 복원 시 자동으로 정리되고, 저장은 원자적으로 이뤄져 마지막 세션이 안전하게 보존됩니다.
