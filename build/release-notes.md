# a8a 업데이트 안내

이번 버전은 실행 로그 창을 더 명확하게 열고 닫을 수 있도록 개선한 Windows 릴리즈입니다.

- 실행 로그 헤더 오른쪽의 `HTML` 버튼 다음에 로그 상태 버튼을 추가했습니다.
- 실행 로그가 닫혀 있을 때는 `열기` 버튼이 표시됩니다.
- 실행 로그가 열려 있을 때는 `닫기(ESC)` 버튼이 표시되어 `Esc` 단축키로 닫을 수 있음을 바로 알 수 있습니다.
- 실행 로그 창이 열린 상태에서 `Esc` 키를 누르면 즉시 실행 로그 창이 닫힙니다.
- `Esc` 닫기 동작은 실행 로그 창이 열린 상태에서만 동작하므로 일반 캔버스 조작에는 영향을 주지 않습니다.
- 영어 UI에서도 같은 위치에 `Open`, `Close (Esc)` 문구를 표시합니다.
- 새 로그 상태 버튼은 기존 `HTML` 리포트 버튼과 같은 높이로 맞춰 하단 실행 로그 헤더의 레이아웃이 어긋나지 않도록 했습니다.
