# a8a 업데이트 안내

이번 버전은 Monaco 편집기 정리 과정에서 발생하던 전역 오류 표시 문제를 수정한 Windows 릴리즈입니다.

- API, SELECT, DATA 설정창의 코드/JSON 편집기를 닫거나 다시 여는 과정에서 `UNHANDLED REJECTION Canceled` 패널이 표시될 수 있던 문제를 수정했습니다.
- Monaco 내부의 정상 취소 신호만 전역 오류 패널에서 제외하므로, 실제 API 실행 오류나 사용자 스크립트 오류는 계속 표시됩니다.
- 전역 오류 패널은 유지해 실제 렌더러 오류를 확인할 수 있습니다.
