# a8a 업데이트 안내

이번 버전은 END 모듈의 HTML/Markdown 리포트에 포함할 실행 정보를 더 세밀하게 선택할 수 있도록 개선한 Windows 릴리즈입니다.

- END 모듈 리포트 설정에 `INPUT`, `OUTPUT`, `PRE REQUEST`, `POST RESPONSE`, `사용된 변수` 포함 여부 체크박스를 추가했습니다.
- 기존 END 모듈 설정은 모두 포함된 상태로 유지되어 이전 프로젝트의 리포트 출력이 갑자기 줄어들지 않습니다.
- 체크를 해제한 항목은 이후 END 모듈이 저장하는 HTML/Markdown 리포트에서 제외됩니다.
- API 모듈의 URL, 요청 헤더, 요청 바디, 응답 바디는 API 실행 증적이므로 `INPUT` 또는 `OUTPUT` 체크 여부와 관계없이 항상 리포트에 포함됩니다.
- HTML 리포트와 Markdown 리포트가 같은 포함 항목 규칙을 따르도록 맞췄습니다.
- 한국어/영어 UI 문구와 README 문서를 새 리포트 설정 동작에 맞게 업데이트했습니다.
